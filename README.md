# 客观信息检索器 (Objective Info Retriever)

一个用于获取和验证客观信息的 TypeScript 工具。

## 功能特性

- 🔍 多源信息检索
- ✅ 事实验证与交叉比对
- 📊 信息可信度评分
- 🌐 支持多种数据源

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build

# 运行
npm start
```

## 项目结构

```
src/
├── index.ts          # 入口文件
├── retriever/        # 信息检索模块
├── validator/        # 信息验证模块
├── types/            # 类型定义
└── utils/            # 工具函数
```
